# Community & Support

TopGun is an open-source project and we welcome contributions, questions, and feedback from the community.

---

## Chat

Join the conversation in real time:

- **Discord** -- [Join our Discord server](https://discord.gg/NDpMG4dmJu)

This is the best place for quick questions, troubleshooting help, and general discussion.

---

## GitHub

- **[GitHub Discussions](https://github.com/topgunbuild/topgun/discussions)** -- Ask questions, share ideas, and discuss features. Use this for longer-form conversations that benefit from threading and search
- **[GitHub Issues](https://github.com/topgunbuild/topgun/issues)** -- Report bugs and request features. Please include reproduction steps and environment details when filing bug reports
- **[Contributing Guide](https://github.com/topgunbuild/topgun/blob/main/CONTRIBUTING.md)** -- Learn how to contribute code, documentation, or tests to TopGun

---

## Getting Help

When asking for help, include:

1. **What you are trying to do** -- describe the goal, not just the error
2. **What happened** -- the exact error message or unexpected behavior
3. **Environment** -- TopGun version, browser/Node.js version, OS
4. **Minimal reproduction** -- a small code snippet that demonstrates the issue

Check the [Troubleshooting guide](/docs/guides/troubleshooting) for solutions to common issues before posting.

---

## Code of Conduct

We are committed to providing a welcoming and inclusive experience for everyone. All participants in TopGun community spaces are expected to:

- Be respectful and considerate in communication
- Welcome newcomers and help them get started
- Focus on constructive feedback
- Respect differing viewpoints and experiences

Harassment, discrimination, and disruptive behavior are not tolerated. Violations can be reported to the project maintainers via GitHub.
