The TopGun server is a Rust binary built on [axum](https://docs.rs/axum) and [tokio](https://tokio.rs). It manages client WebSocket connections, CRDT synchronization, Merkle tree delta sync, full-text search, clustering, and durable PostgreSQL storage.

## Configuration

The Rust server is configured via two primary mechanisms: **environment variables** for runtime settings, and **Rust structs** for programmatic embedding.

### Environment Variables

The server reads three environment variables at startup: `PORT`, `TOPGUN_ADMIN_DIR`, and `RUST_LOG`. See the <a href="/docs/reference/cli#environment-variables">CLI Reference</a> for the full table with descriptions and defaults.

## PostgreSQL Storage

The `PostgresDataStore` provides durable write-through persistence via `sqlx::PgPool`. Every mutation is committed to PostgreSQL before the async method returns.

## Graceful Shutdown

The server supports graceful shutdown via SIGTERM or SIGINT:

1. Health state transitions to **Draining**
2. All active WebSocket connections receive a Close frame
3. In-flight requests are given up to 30 seconds to complete
4. Health state transitions to **Stopped**
