# Tutorials

Learn TopGun by building real projects. Each tutorial walks you through a complete application from start to finish.

---

## Available Tutorials

### Todo App

**Difficulty:** Beginner | **Time:** ~15 minutes

Build a React todo application with offline-first data and real-time sync. You will learn how to:

- Set up a Vite + React project with TopGun
- Define an LWWMap for todo items
- Use `useMap` and `useMutation` React hooks
- See offline capability in action
- Observe real-time sync between browser tabs

[Start the Todo App Tutorial](/docs/tutorials/todo-app)

---

## Prerequisites

Before starting any tutorial, make sure you have:

1. **Node.js 18+** and **pnpm** installed
2. **The TopGun Rust server** running locally (see [Quick Start](/docs/quick-start) for setup)
3. Basic familiarity with React and TypeScript
